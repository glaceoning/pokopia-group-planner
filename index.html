        How Scores Work

        • For each existing group member: habitat contributes +1 if ideal habitat matches, -1 if habitat is in conflict (i.e. dark, bright), and 0 otherwise.
        • Favorites contribute +n where n is the number of shared favorites with that member.
        • Let slider value be a (from -1 to +1): combined = (1 - a) * habitat_score + (1 + a) * favorites_score. We may update this formula later as game mechanics become clearer.
        • Group totals are shown in two modes: stacked counts full overlap build-up (k - 1 + ... + 1), while simple counts k - 1 for k Pokemon sharing a feature. Conflicting habitat pairs subtract 1 in both modes.
